📱 수동으로 추가하기

위 버튼이 작동하지 않는 경우, 아래 URL을 복사하여 수동으로 캘린더에 추가하세요:

🔸 구글 캘린더 수동 추가:

  1. 구글 캘린더(calendar.google.com) 접속
  2. 왼쪽 메뉴에서 "다른 캘린더" 옆 "+" 버튼 클릭
  3. "URL로 추가" 선택
  4. 위 URL 붙여넣기 후 "캘린더 추가" 클릭

🔸 네이버 캘린더 수동 추가:

  1. 네이버 캘린더(calendar.naver.com) 접속
  2. "캘린더 관리" → "캘린더 추가" 클릭
  3. "외부 캘린더 가져오기" 선택
  4. 위 URL 입력 후 "추가" 클릭
